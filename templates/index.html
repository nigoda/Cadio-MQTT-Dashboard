account_tree Topic Structure 
{prefix}/{type}/{account_id}/{serial_channel}/{action}

Examples:
  homeassistant/light/d11cdb3057224ee6/2CF4327CA967_6/set      → Send command
  homeassistant/light/d11cdb3057224ee6/2CF4327CA967_6/state    → Read state
  homeassistant/light/d11cdb3057224ee6/2CF4327CA967_6/config   → Discovery config
  homeassistant/switch/d11cdb3057224ee6/A4CF12F03246_0/set     → Send command

Actions:
  /config         Discovery config (auto-detected)
  /state          Current state (read-only, published by device)
  /set            Send commands (write)
  /availability   Online status (YES / NO)
